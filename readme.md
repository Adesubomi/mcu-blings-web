#Blings

### Installation
* Clone the project
```git clone blings```
* Install composer dependencies ```composer install```
* Run database migrations ```php artisan migrate```
* Seed initial data for the database table ```php artisan db:seed```
* Install security keys for laravel-passport ```php artisan passport:install```
* Install node.js dependencies ```npm i```
* Compile front-end assets ```npm run prod```
* Serve your application ```php artisan serve```
